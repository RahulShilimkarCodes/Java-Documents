# 3. Java Abstraction Cheat Sheet

---

```text id="cs1"
Interface:
- 100% abstraction (before Java 8)
- Supports multiple inheritance
- Used in frameworks

Abstract Class:
- Partial abstraction
- Can have concrete methods

Key Idea:
- "What to do" → Interface
- "How to do" → Implementation hidden
```

# 6. 100 Interview Questions with Answers

---

## BASIC (1–20)

---

**Q1:** What is abstraction?
**A:** Hiding implementation and showing only functionality.

---

**Q2:** How is abstraction achieved?
**A:** Using interfaces and abstract classes.

---

**Q3:** Can we instantiate abstract class?
**A:** No.

---

**Q4:** Can interface have methods?
**A:** Yes.

---

**Q5:** Difference between abstraction and encapsulation?
**A:** Abstraction hides logic, encapsulation hides data.

---

**Q6:** Why abstraction is used?
**A:** To reduce complexity.

---

**Q7:** Can interface have variables?
**A:** Yes, static final only.

---

**Q8:** Can abstract class have constructor?
**A:** Yes.

---

**Q9:** Can interface extend class?
**A:** No.

---

**Q10:** Can class implement multiple interfaces?
**A:** Yes.
